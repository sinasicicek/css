Display :Bir elementin nasıl görünecegini yada görünmeyecegini belirler

Visibilty :Bir elementin görünür olup olmadıgını belirler

margin : en dıştaki katmandır auto özelligi verilirse sayfanın tam orta noktaya göre hizalanır ve margin dışdan yani tag dışından konum alır
border: html taglarını çercevesidir

Buradaki tag inceleyelim

Html ögelerini görüntülenip/görüntülenmeyecegini yada nasıl görünücegini belirler. Normal şartlarda Div etiketi block özellige yani tüm satırı kaplayacak şekilde ayarlıdır, bu durumu degiştirmek için display kullanılır
display:none

Bu özellik atanan html elemanı render edilmez ve ekranda görünmez


display:inline

Bu özellik atanan html elemanı block olan taglar inline olur.Örnegin li etiketleri alt alt degil yana yana dizilir.yani tek satırda hepsi ayrı olur
