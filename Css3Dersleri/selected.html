    BÖLÜM 2 (Seçiciler)

  • Class ile Seçme


    Class ile seçmek için css dosyası içinde .metinC { color :red ; }

  • Seçilen Class içinden Tag seçme


    lar içinde sadece h1 yada p taglarına özel bir özellik atanmak istenirse yani sadece p tagına
    p.metinC { // Buradaki özellikler metinC classına ait p taglarını etkileyecektir }

  • Birden fazla class özelligi atama


    .özellik1 { color :red ;} .özellik2 { background : yellow ; } <p class="özellik1 özellik2">Ozellikleri birlestiriyoruz </p>

  • Tüm tagları seçme


    tüm taglara etkilemsi için * operatörü kullanılır * { text-align: center; color: blue; }

  • id degeri ile seçme

    degeri ile seçmek için
    #metin { color : red ;}

  • Toplu Olarak Seçim Yapma


    İstenirse toplu olarak seçim yapılabilir p,h1 p , h1 { text-align: center; color: blue; }
  • Taglar içindeki attribute özellikler verilebilir Html tarafında <p title="Sinasi"></p> css tarafında p[title] { border : 2px ;}
