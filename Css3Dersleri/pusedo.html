Kullanımı 2 ayrılır :
  • Pseudo Element : Herhangi bir elemanın bir parçasını seçmek için kullanılır
  • Pseudo Class : Elemanın alt elemanlarını seçerken kullanılır.Yani Bir alt öge seçi mi yapılır

Eleman Seçme

Burada bulunan p etiketlerinin için yazılan bu yazının üstüne gelindiginde rengini degiştirmek için gerekli kod yapısı

p :hover { color : hotpink } // Burada p etiketine sahip hangi elemanın üstüne mouse ile gelirsen rengi degişir

Elemanların classlarına ulaşma

Burada p etiketlerinden class adı hilal olanların ilk satrıları rengi degişicektir

p.hilal::first-line { color:lightseagreen; } <p class="hilal"></p>

Bir tag için oluşan olay sonucunda başka bir tag etkilenebilir

span { display: none; background-color:yellow ; } div:hover span { display : block ; }
Herhangi bir div alanına geldiginde Bu alan rengi degişicektir
