Position özelligi bir element için kullanılan konumlandırma yöntemi tipidir, 5 konumlandırma şekli vardır

  • Static
  • relative
  • fixed
  • absolute
  • sticky

Sticky Konumlandırma

Kaydırma cubugu aşagıya cekildiginde sayfanın en üst kısmında kalıcaktır

div.sticky { position: -webkit-sticky; position: sticky; top: 0; padding: 5px; background-color: #cae8ca; border: 2px solid #4CAF50; } </style> Kullanımı :
Burada sticky var kaydırma cubugunu kullan ve gör ama önce boşlukla sayfanınbüyütelim

Static Konumlandırma

Html tagları varsayılan olarak gelir ve margin yada pading ile degiştirilemez.

Relative Konumlandırma

İstenilen herhangi bir yönden px olarak konumlandırmamızı saglar istenirse sagdan 10px yada yukarıdan 20px yer degiştirilebilir kullanımı :

<style> div.relative { position: relative; left: 30px; background-color: cornflowerblue; } </style>

Yukarıdaki kodların cıktısı :


Burada relative uygulandı ve soldan 30px uzaklaştırıldı

fixed Konumlandırma

Belirlenen Konum neresi ise orada konumlandırılır ve sayfa kaydırılsa dahi yeri degişmez

.fixd { position:fixed ; bottom : 0 ; right : 0 ; width : 300px; border : 3px solid red; } Bu kodun cıktısı
Burda fixed ile alakalı cıktı bulunuyor

Z İndex

Bir ögenin önüne ,arkasına yanıan gibi konumlandırmaları yapamk için kullanılır

img { position: absolute; left: 0px; top: 0px; z-index: -1; }

Paylaşılan resim arka font olusacaktır.
position özelligi absolute olursa sol üst köşede olusur
