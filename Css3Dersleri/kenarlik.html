• border-style: solid; kenarlık türünü seçiyoruz
  • border-width: 5px; kenarlıgın genişligi
  • border-width: 20px 5px 10px 5px ; sırasıyla üstten,sagdan,alttan,soldan boşluklar istenirse tek yada iki tane deger verilebilir
  • border-color: red; kenarlıgın rengini verir istenirse 4 renk verilerek sırayla üstten,sagdan,alttan soldan renleri verbilriz
  • CSS'de, kenarlıkların her birini (üst, sağ, alt ve sol) belirtmek için özellikler de vardır: p { border-top-style: dotted; border-right-style: solid; border-bottom-style: dotted; border-left-style: solid; }
  • Ayrıca yalnızca bir taraf için tüm kenarlık özelliklerini belirleyebilirsiniz: p { border-left: 6px solid red; background-color: lightgrey; }
  • Radius ile kenarlık yarı capı belirlenir p { border: 2px solid red; border-radius: 5px; }
