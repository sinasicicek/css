Pseudo class

Secilen element içindeki tag olacakları belirleyebiliriz.Örnegin div etiketinin üstüne geldigmizde p taglarına olacakları belirlemek için :

p { display: none; background-color: yellow; padding: 20px; } div:hover p{ div üzerine geliindiginde p tag için olacaklar } İslem gerçekleşti
p::after { content: " - yeni içerik eklendi"; }

Buradaki alana daaha sonradan eklenecek metin yukarıdaki

  • p::after : p etiketinden sonra eklenicek
  • p::before : p etiketinden önce eklenicek
  • p::first-letter: p etiketinin degerinin ilk harfine olacaklar
